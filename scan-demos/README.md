# Scan demo videos

Looping ~5-second clips shown in the corner thumbnail during capture so
customers can mimic the motion.

- `turning.mp4` — finger pivoting on its tip with card at table edge.
- `ccurve.mp4` — camera orbiting an upright finger with nail touching card edge.

These are currently **placeholders** (5s of black frames). The store owner
records the real content separately and drops it here. Files are served from
`/scan-demos/` and referenced in `CaptureVideoStep.tsx`.
